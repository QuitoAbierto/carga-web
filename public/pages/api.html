Documentación de la API


En este sitio encontraras información de la API de Quito Abierto. Puedes ver los datasets existentes y los endpoints disponibles.


Datasets


En este momento existen dos datasets:

  • Paradas
  • Rutas


Paradas /api/parada


Para obtener la información de paradas se pueden usar los siguientes endpoints:

endpoint metodo descripción respuesta
/api/parada GET Este endpoint se utiliza para obtener todas las paradas disponibles. La respuesta es una lista de todas las paradas en formato JSON.

Ejemplo:

RESPUESTA: [{"location": {"lat": 100, "lng": 100}, "geoJSON": {"type": "Feature", "properties": {"name": "Parada 1", "description": "Cerca de la carolina"}, "geometry": {"coordinates": [100, 100], "type": "Point"}}, "name": "Parada 1", "description": "Cerca de la Carolina", "type": "parada"}]
/api/parada/cercana GET Este endpoint se utiliza para retornar la parada más cercana a la unicación que se provee. La respuesta es la parada más cercana que se puede encontrar.

Ejemplo:

GET /api/parada/cercana -d '{"location": {"lat": 100, "lng": 200}}'

RESPUESTA: {"location": {"lat": 100, "lng": 100}, "geoJSON": {"type": "Feature", "properties": {"name": "Parada 1", "description": "Cerca de la carolina"}, "geometry": {"coordinates": [100, 100], "type": "Point"}}, "name": "Parada 1", "description": "Cerca de la Carolina", "type": "parada"}

Rutas /api/ruta


Para obtener la información de paradas se pueden usar los siguientes endpoints:

endpoint metodo descripción respuesta
/api/ruta GET Este endpoint se utiliza para obtener todas las rutas disponibles. La respuesta es una lista de todas las rutas en formato JSON.

Ejemplo:

RESPUESTA: [{"location": {"lat": 100, "lng": 100}, "geoJSON": {"type": "Feature", "properties": {"name": "Parada 1", "description": "Cerca de la carolina"}, "geometry": {"coordinates": [100, 100], "type": "Point"}}, "name": "Parada 1", "description": "Cerca de la Carolina", "type": "parada"}]
/api/ruta/<nombre> GET Este endpoint se utiliza para retornar una ruta por nombre. La respuesta es la ruta con todos sus puntos.

Ejemplo:

GET /api/ruta/ruta1

RESPUESTA: {"location": {"lat": 100, "lng": 100}, "geoJSON": {"type": "Feature", "properties": {"name": "Parada 1", "description": "Cerca de la carolina"}, "geometry": {"coordinates": [100, 100], "type": "Point"}}, "name": "Parada 1", "description": "Cerca de la Carolina", "type": "parada"}
